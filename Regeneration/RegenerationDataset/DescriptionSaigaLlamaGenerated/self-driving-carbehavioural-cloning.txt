В проекте по поведенческой клонированию автомобилей без водителя используется датасет, состоящий из файлов с изображениями дорожного полотна и условий. Метод сбора данных отсутствует, но структура содержит информацию о параметрах и настройках модели нейронной сети в файле `model_config`, а также пути к данным для обучения. Основной файл README.md описывает процесс создания автомобильного беспилотника, включая таблицы с проблемами и решениями, структурой нейронных сетей и их использованием для обучения модели. Файл `backend` позволяет настраивать параметры обучающего процесса и указать пути к данным для обучения. Файлы `self-driving-carbehavioural-cloning` и `keras_version`, вероятно, используются только для работы с программой, но их функции неизвестны. Таким образом, проект предоставляет инструменты для настройки обучающего процесса, загрузки данных и описания разработки беспилотников, без информации о потенциальных юзкейсах.
