Данные для рекомендательной системы были собраны из открытых данных пользователей KuaiRand-Pure без ограничений, включая информацию о предпочтениях, покупках, просмотрах и действиях в сети. Общее количество собранных данных составляет 1,4 миллиона записей о выборе продукта, которые использовались для обучения модели. Структура данных не указана, а потенциальные юзкейсы отсутствуют в предоставленной информации.
