Датасет Women Apparel Recommendation Engine (amazon.com) содержит информацию о женской одежде, собранную из интернета для создания рекомендательных систем на Amazon.com. Строгая структура данных представлена в файле tops_fashion.json, включающем словари с ключевыми полями, такими как SKU, ASIN, product_type_name, formatted_price и другие характеристики. Этот датасет может быть использован для разработки систем рекомендаций, автоматизации покупок и улучшения пользовательского опыта на электронных магазинах, предоставляя потенциальные юзкейсы для разработчиков.
