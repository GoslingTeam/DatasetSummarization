Датасет содержит три файла, описывающих информацию о рейтингах книг и пользователях. Файл `rating.csv` включает три столбца: `user_id` (идентификатор пользователя), `item_id` (идентификатор книги) и `rating` (оценка книги пользователем). Этот файл предоставляет данные о том, какие книги были оценены пользователями и какие оценки они получили. Файл `train.csv` содержит только два столбца: `user_id` и `item_id`, что указывает на пользователей и книги, с которыми они взаимодействовали, без информации о самих оценках. Файл `test.csv` содержит только столбец `user_id`, предоставляя список пользователей, для которых, вероятно, предполагается предсказать рекомендации или оценки.
