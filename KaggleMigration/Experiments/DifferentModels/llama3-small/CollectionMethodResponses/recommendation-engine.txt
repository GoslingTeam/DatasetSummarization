Данные датасета Recommendation Engine были собраны для рекомендации вопросов, которые программист должен решать. Это множество данных содержит информацию о задачах (problem_data.csv), которая включает в себя ID проблем, уровень сложности, количество баллов и теги. Крім того, в датасете представлены данные об альтернативных ответах пользователей на эти задачи (submission.csv) и тестовые ответы (test_submissions_NeDLEvX.csv). Данные были собраны для создания системы рекомендации, которая поможет программистам найти наиболее подходящие задачи для их уровня навыков.
