Данный датасет содержит информацию о результатах нескольких экспериментов, представленных в виде записей. Каждая запись содержит идентификативный код пользователя, идентификативный код задачи и результат каждой Experimentaции. Структура датасета простая - она состоит из нескольких столбцов, таких как ID, user_id, problem_id и prob_id, которые содержат идентификативные данные для каждого объекта.
