Данный датасет "YTS movie dataset" содержит информацию о фильмах, которые доступны на платформе YouTube. Он может быть использован для различных целей, например, для анализа популярности фильмов, создания системы рекомендаций, или для изучения трендов в кинематографе. Данные о фильмах в датасете включают название, год выхода, Рейтинг IMDb, постера, жанры, режиссёра, актеров и список похожих фильмов. Этот датасет может быть использован для создания различных аналитическихinsightów и для построения прогнозных моделей.
