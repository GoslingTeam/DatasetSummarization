Данный датасет содержит информацию оNear-Earth Objects (NEOs) от NASA. Он содержит множество данных о каждом NEO, например, его название, размер, орбиту и другие характеристики. Этот датасет может быть использован для различных целей, например, для изучения свойств NEOs, для прогнозирования их движения, или для разработки алгоритмов для опознания потенциально опасновых объектов.
