Вот dataset «Hotel Recommendation Dataset» содержит информацию о гостиницах и бронировании в отелях. Датасет состоит из одного файла train.csv, который содержит 18 столбцов с различной информацией о пользователях, поисках и бронированиях.

Столбцы делятся:

* site_name - имя сайта
* posa_continent - континент, где находится пользователь
* user_location_country - страна, где находится пользователь
* user_location_region - регион, где находится пользователь
* user_location_city - город, где находится пользователь
* user_id - уникальный идентификатор пользователя
* is_mobile - признак мобильного устройства
* is_package - признак бронирования пакета
* channel - канал бронирования
* srch_adults_cnt - количество взрослых в поиске
* srch_children_cnt - количество детей в поиске
* srch_rm_cnt - количество номеров в поиске
* srch_destination_id - идентификатор места назначения
* srch_destination_type_id - тип места назначения
* is_booking - признак бронирования
* cnt - количество
* hotel_continent - континент, где находится отель
* hotel_country - страна, где находится отель
* hotel_market - рынок, где находится отель
* hotel_cluster - кластер, где находится отель

Информация в датасете представлена в виде таблицы с разделителями запятыми. Каждая строка представляет собой один пример бронирования гостиницы.
