Данный датасет предназначен для прогнозирования покупки страхового продукта на основе истории покупок клиента. Он содержит информацию о клиенте, его поведении при покупке и факторах, влияющих на принятие решения о покупке страхового продукта.

Возможные варианты использования датасета:

1. Обучение моделей машинного обучения для прогнозирования покупки страхового продукта на основе истории покупок клиента.
2. Анализ данных для поиска факторов, влияющих на принятие решения о покупке страхового продукта.
3. Разработка рекомендательной системы для предложения подходящих страховых продуктов клиентам на основе их истории покупок.

Датасет содержит информацию о клиенте, включая в себя следующие столбцы:

* customer_ID - уникальный идентификатор клиента
* shopping_pt - точка времени покупки
* record_type - тип записи (например, 0 - покупка, 1 - просмотр)
* day - день недели
* time - время дня
* state - штат или регион проживания клиента
* location - местоположение клиента
* group_size - количество членов семьи клиента
* homeowner - наличие дома у клиента
* car_age - возраст автомобиля клиента
* car_value - стоимость автомобиля клиента
* risk_factor - фактор риска для клиента
* age_oldest - возраст старшего члена семьи клиента
* age_youngest - возраст младшего члена семьи клиента
* married_couple - наличие брака у клиента
* C_previous - предыдущий страховой продукт
* duration_previous - продолжительность предыдущего страхового продукта
* A, B, C, D, E, F, G - факторы, влияющие на принятие решения о покупке страхового продукта
* cost - стоимость страхового продукта
