Датасет "Tehran renting" представляет собой столбцово-табличный набор данных, который содержит информацию о различных квартирах и виллах для аренды в Тегеране. Структура каждого ряда включает в себя следующие параметры: количество депозита, цена аренды, этаж, площадь, возраст дома, количество спалей, есть ли лифт, парковка, хранение багажа и время публикации объявления.

Каждый столбец имеет уникальное имя (например, "deposit", "rent", "floor" и т.д.), что облегчает работу с данными для начинающих пользователей. В целом, данный набор данных может быть использован для анализа рынка недвижимости в Тегеране и для поиска подходящей квартиры или виллы для аренды.
