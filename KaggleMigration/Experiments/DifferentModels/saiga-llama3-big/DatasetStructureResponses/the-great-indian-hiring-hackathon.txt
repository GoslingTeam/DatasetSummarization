Датасет состоит из большого количества записей с информацией о заказах клиентов в интернет-магазине. Каждая строка представляет собой отдельный заказ и содержит информацию по следующим пунктам:

* InvoiceNo - уникальный номер заказа;
* StockCode - артикул товара;
* Description - краткое описание товара;
* Quantity - количество заказанных единиц товара;
* InvoiceDate - дата оформления заказа;
* UnitPrice - цена одной единицы товара на момент оформления заказа;
* CustomerID - идентификатор клиента, который сделал заказ;
* Country - страна, в которой находится клиент.

Кроме того, в датасете есть информация о количестве каждой строки (rowid), но эта информация не играет существенной роли для анализа данных.
