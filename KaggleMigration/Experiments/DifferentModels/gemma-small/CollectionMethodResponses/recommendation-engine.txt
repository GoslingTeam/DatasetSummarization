Данные из датасета были собраны из нескольких источников. Первые 1024 байт съдържали ID пользователя, ID проблемы и ID задачи. 
Следующие 1024 байт содержали данные для проблемы, а следующие 1024 байт содержали данные для задачи. 
Наконец, оставшая часть файла содержит данные, использованные для создания модели recommend engine.
