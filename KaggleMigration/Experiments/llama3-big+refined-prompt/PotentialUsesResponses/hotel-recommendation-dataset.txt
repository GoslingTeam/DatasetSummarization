Данный файл - это часть датаcета "Hotel Recommendation Dataset", которая содержит информацию о бронировании отелей, пользовательских предпочтениях и других параметрах для рекомендации отелей.

В файле train.csv представлены следующие столбцы:

* site_name: имя сайта
* posa_continent: континент, где находится пользователь
* user_location_country: страна, где находится пользователь
* user_location_region: регион, где находится пользователь
* user_location_city: город, где находится пользователь
* user_id: идентификатор пользователя
* is_mobile: признак мобильного устройства
* is_package: признак пакета бронирования
* channel: канал бронирования (например, поискомый запрос)
* srch_adults_cnt: количество взрослых людей в поиске
* srch_children_cnt: количество детей в поиске
* srch_rm_cnt: количество комнат в поиске
* srch_destination_id: идентификатор места назначения
* srch_destination_type_id: тип места назначения (например, город, страна)
* is_booking: признак бронирования
* cnt: количество броней
* hotel_continent: континент, где находится отель
* hotel_country: страна, где находится отель
* hotel_market: рынок, где находится отель
* hotel_cluster: кластер отелей

Данные столбцы содержат информацию о пользовательских предпочтениях и параметрах бронирования, что может быть использовано для рекомендации отелей на основе Machine Learning алгоритмов.
