Структура датасета состоит из двух файлов: "Data\CooperUnion\anime-recommendations-database\anime.csv" и "Data\CooperUnion\anime-recommendations-database\rating.csv". Файл "anime.csv" содержит информацию о различных аниме, включая идентификатор аниме, название, жанр, тип, количество эпизодов, рейтинг и количество членов. Файл "rating.csv" содержит информацию о рейтингах пользователей для различных аниме, включая идентификатор пользователя, идентификатор аниме и рейтинг.
