Файл anime.csv содержит информацию о различных аниме, включая их идентификатор, название, жанр, тип (фильм или сериал), количество эпизодов и рейтинг пользователей. Файл rating.csv содержит данные о рейтинге пользователей для каждого аниме. В каждом файле есть несколько столбцов: идентификатор пользователя, идентификатор аниме и оценка.
