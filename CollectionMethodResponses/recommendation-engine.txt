Было собрано большое количество данных пользователей с их решениями задач и соответствующими проблемами, которые используются в автоматическом системе рекомендаций «Сайга» для определения вероятности успешного решения пользователем проблемы.

Данные содержатся в файле Data/ashishvaya/recommendation-engine/recommendation-engine/test_submissions_NeDLEvX.csv и представляют собой информацию о пользователях (ID, user_id), проблемах (problem_ид) и вероятности их успешного решения. Файл состоит из большого количества строк, каждая из которых содержит данные о пользователе и его решении конкретной проблемы.

Все данные в файле были собраны для использования в системе рекомендаций «Сайга» с целью предоставления пользователю наиболее подходящих вариантов решения определенных задач.
