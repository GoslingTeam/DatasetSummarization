Датасет "Topic Modeling for Research Articles" содержит информацию о статьях по научным исследованиям и связанных с ними тематических ключевых словах (tags). Он состоит из четырех файлов: Sample.csv, Tags.csv, Test.csv и Train.csv.

Sample.csv - это примерный файл данных, который не используется в алгоритмах машинного обучения. Он содержит только идентификатор статьи и нулевые значения для всех остальных столбцов.

Tags.csv - это файл с тематическими ключевыми словами (tags) для каждой статьи. Он содержит список всех возможных тегов, которые могут быть применены к статье.

Test.csv и Train.csv - это файлы с данными для обучения и проверки моделей машинного обучения. Они содержат идентификатор статьи, полное описание (ABSTRACT) статьи и его жанр (Computer Science, Mathematics, Physics, Statistics). При этом файл Test.csv используется для оценки точности модели во время тестирования, а файл Train.csv - для ее обучения.

Нет информации о том, как данные из датасета были собраны.
