Датасет "Recommendation Engine" содержит несколько файлов, каждый из которых имеет свою функцию. Основной файл данных `problem_data.csv` содержит информацию о задачах, таких как идентификатор задачи (`problem_id`), уровень сложности (`level_type`), количество очков (`points`) и теги (`tags`). Файл `submission.csv` фиксирует попытки пользователей решить задачи, указывая идентификатор пользователя и задачи, а также количество попыток (`attempts_range`). Файл `test_submissions_NeDLEvX.csv` предоставляет тестовые данные, связывая идентификаторы пользователей и задач, а также содержит идентификатор тестовой записи (`ID`). Файл `sample_submissions_wbscxqU.csv` не содержит данных о задачах и попытках, только заголовки.
