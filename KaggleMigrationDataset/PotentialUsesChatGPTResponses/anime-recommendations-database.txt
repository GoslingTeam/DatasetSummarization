Датасет "Anime Recommendations Database" содержит рекомендации аниме от 76,000 пользователей с сайта myanimelist.net. Он включает два основных файла: `anime.csv`, который описывает характеристики аниме (ID, название, жанр, тип, количество серий, рейтинг и количество участников), и `rating.csv`, который содержит данные о рейтингах, выставленных пользователями (ID пользователя, ID аниме и рейтинг). Этот набор данных может быть использован для различных задач, таких как создание рекомендательных систем для аниме, анализ популярных жанров и типов аниме, исследование влияния количества участников на рейтинг, а также для построения статистических моделей предпочтений пользователей. Кроме того, данные могут служить основой для изучения тенденций в аниме индустрии и выявления ключевых факторов, влияющих на популярность аниме.
