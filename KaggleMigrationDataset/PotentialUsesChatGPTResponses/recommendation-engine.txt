Датасет **Recommendation Engine** предназначен для рекомендаций вопросов, которые программистам следует решить, и включает в себя несколько файлов с различной информацией. Основной файл данных (`problem_data.csv`) содержит информацию о различных задачах, таких как их идентификаторы, уровень сложности, количество очков за решение и теги, которые описывают темы задач. Файл `submission.csv` предоставляет данные о попытках пользователей решить задачи, указывая, сколько раз каждая задача была решена. Файл `test_submissions_NeDLEvX.csv` связывает идентификаторы пользователей с конкретными задачами, которые они пытались решить. Эти данные могут быть использованы для построения систем рекомендаций, которые помогут пользователям выбирать задачи в зависимости от их уровня подготовки и интересов, а также для анализа успеха пользователей в решении задач и оценки сложности различных задач.
