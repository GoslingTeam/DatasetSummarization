Датасет "Bitcoin Network On-Chain Blockchain Data" включает в себя несколько файлов с различными типами данных, связанными с блокчейном Биткоин. Основной файл данных — `blockchain_dot_com_daily_data.csv` — содержит ежедневные значения для множества метрик, таких как размер мемпула (mempool_size), скорость транзакций (transaction_rate), рыночная капитализация (market_cap_usd), средний размер блока (average_block_size), рыночная цена (market_price_usd), объем торгов на биржах (exchange_volume_usd), среднее время подтверждения транзакций (average_confirmation_time), хэшрейт сети (hash_rate), сложность (difficulty), доходы майнеров (miners_revenue) и общие комиссии за транзакции (total_transaction_fees). Кроме того, в датасете есть файл `blockchain_dot_com_half_hourly_data.csv`, который предоставляет полудневные данные для транзакционной скорости, размера мемпула и рыночной капитализации. Также присутствует файл `look_into_bitcoin_address_balances_data.csv`, который содержит информацию о балансе адресов Биткоин с различными уровнями средств и рыночной ценой. Описание столбцов и данных, представленных в файлах, помогает глубже понять динамику и характеристики сети Биткоин на протяжении времени.
