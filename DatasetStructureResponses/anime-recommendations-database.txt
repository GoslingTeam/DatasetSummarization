Датасет Anime Recommendations Database представляет собой набор данных по аниме, собранных от 76 тысяч пользователей MyAnimeList.net. Структура этого датасета состоит из двух файлов: anime.csv и rating.csv.

Файл anime.csv содержит информацию о каждом аниме, включая его ID, имя, жанры, тип (фильм или сериал), количество эпизодов и рейтинг. 

Файл rating.csv представляет собой таблицу связей между пользователями и их оценками аниме. Каждая строка в файле содержит ID пользователя, ID аниме, которое оценивается этим пользователем, и оценку, данную пользователем этому аниме.

В целом, данный датасет предоставляет возможность провести рекомендации аниме для пользователей на основе их предпочтений и оценок других пользователей.
